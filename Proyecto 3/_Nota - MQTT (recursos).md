# Recursos MQTT (Proyecto 3 — IoT)

Para la comunicación entre los controladores embebidos y el sistema de administración se sugiere **MQTT** (protocolo de mensajería publish/subscribe, liviano, típico en IoT).

- **MQTT (sitio oficial del protocolo):** https://mqtt.org/
- **HiveMQ (broker MQTT):** https://www.hivemq.com/
